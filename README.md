# Repositório de Projetos em Estrutura de Dados 2

## Descrição

Bem-vindo ao meu repositório de projetos em Estrutura de Dados 2! Este repositório contém uma coleção de projetos e trabalhos práticos desenvolvidos como parte do meu curso de Sistemas de Informação.

## Sobre Mim

Sou um estudante de Sistemas de Informação apaixonado por programação e resolução de problemas complexos. Minha ênfase está na construção de algoritmos eficientes e na implementação de estruturas de dados robustas para resolver uma variedade de desafios de computação.

## O que você encontrará aqui

- Implementações em C de diversos tipos de estruturas de dados, incluindo listas ligadas, árvores, grafos e filas de prioridade.
- Soluções para problemas práticos e desafios de programação que exploram o uso dessas estruturas de dados em cenários do mundo real.
- Projetos que demonstram meu domínio de conceitos avançados de Estrutura de Dados, como algoritmos de ordenação e busca, manipulação de grafos e otimização de desempenho.
